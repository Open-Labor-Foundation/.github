# Open Labor Foundation

*What you know lifts us all.*

Shared knowledge infrastructure for workers — built by workers, maintained by the community, available to everyone. A catalog of specialist expertise any worker, collective, or platform can draw from, without a platform intermediary taking a cut.

---

## The stack

| Repo | What it does |
|---|---|
| [labor-commons](https://github.com/Open-Labor-Foundation/labor-commons) | The shared catalog — NAICS-grounded specialist definitions for every profession and domain |
| [commons-crew](https://github.com/Open-Labor-Foundation/commons-crew) | Personal assistant backed by the catalog — any task, any domain, no retainer |
| [commons-board](https://github.com/Open-Labor-Foundation/commons-board) | Organizational AI for any group of workers — full governance structure, three autonomy modes |
| [commons-idea](https://github.com/Open-Labor-Foundation/commons-idea) | Turns any idea into a working local deliverable — plain language in, running software out |
| [commons-keeper](https://github.com/Open-Labor-Foundation/commons-keeper) | Autonomous quality loop — keeps the catalog accurate and current |
| [commons-specs](https://github.com/Open-Labor-Foundation/commons-specs) | Community library of idea records contributed back by workers |
| [commons-artifacts](https://github.com/Open-Labor-Foundation/commons-artifacts) | Community library of working deliverables — one-click local apps shared back for anyone to use |

---

## Contributing

The most valuable contributions come from people who know their field — any trade, profession, or specialized domain. No technical background required.

Use [commons-idea](https://github.com/Open-Labor-Foundation/commons-idea) to capture your expertise or build something new. Contribute specialist knowledge to [labor-commons](https://github.com/Open-Labor-Foundation/labor-commons), written records to [commons-specs](https://github.com/Open-Labor-Foundation/commons-specs), and working deliverables to [commons-artifacts](https://github.com/Open-Labor-Foundation/commons-artifacts).

A streamlined contribution interface is actively in development. See [CONTRIBUTING.md](https://github.com/Open-Labor-Foundation/open-labor-foundation/blob/main/CONTRIBUTING.md) for the full picture.
